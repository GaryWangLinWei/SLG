# 新用户教学 / 入门引导 设计说明书

**日期:** 2026-05-28  
**状态:** 设计中

---

## 概述

首次激活后进入教学页面，覆盖完整使用流程。用户可跳过，后续从导航栏"帮助"按钮重新查看。

## 设计

### 教学步骤（5 步）

| 步 | 标题 | 内容 | 快捷按钮 |
|----|------|------|----------|
| 1 | 激活授权 | 介绍激活码机制、到期续费、在线购买链接 | 无（已激活才能看到此页） |
| 2 | 配置模拟器 | ADB 连接模拟器、选择账号、多账号管理 | "去配置模拟器 →" → `/accounts` |
| 3 | 校准坐标 | 建筑位置校准、不同分辨率适配 | "去校准坐标 →" → `/config` |
| 4 | 选择功能 | 采集/训练/科技等功能开关说明 | "去选择功能 →" → `/` |
| 5 | 开始运行 | 点击开始后 OCR 自动识别循环运行 | "开始使用 →" → `/` |

### 触发逻辑

- 首次激活后，`LicenseGate` 检测 `localStorage` 中 `tutorial-seen` 键
- 若不存在 → 自动跳转 `/help`（教学页）
- 用户点"跳过"或"开始使用" → 设置 `tutorial-seen: true`，跳转 `/`
- 从导航栏"帮助"进入 → 不触发自动跳转，仅展示内容

### 导航栏入口

- NavBar 右侧（已激活状态下）添加"帮助"链接 → `/help`

### 页面结构

```
TutorialPage（全屏页面）
├── 退出按钮（右上角 X，关闭回到首页）
├── 步骤进度条（5 个圆点，当前步骤高亮）
├── 内容区
│   ├── 插图/截图区域
│   ├── 步骤标题
│   ├── 文字说明
│   └── 快捷跳转按钮（可选，跳转到对应配置页）
└── 底部导航
    ├── 上一步（第 1 步隐藏）
    ├── 下一步（第 5 步变为"开始使用"）
    └── 跳过（直接进入首页）
```

### 状态存储

- `localStorage` key: `tutorial-seen` → `"true"` | 不存在
- 仅在教学页内设置，外部只读
